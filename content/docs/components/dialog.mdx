## Instalation

#### 1. Install dependencies:

```sh
npm install @radix-ui/react-dialog @radix-ui/react-visually-hidden tailwindcss-animate
```

<br />

#### 2. Configure your `tailwind.config.ts` add:

```sh
{
  content: [...],
  theme: [...],

  // add plugin
  plugins: [require("tailwindcss-animate")],
}
```
